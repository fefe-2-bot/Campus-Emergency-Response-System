# Campus Emergency Response System

A Django-based web application for managing campus emergencies with role-based access control and real-time notifications.

## Features

- **Role-based Authentication**: Students, Fire Department, Health Department, Social/Bullying Center, and Admin users
- **Incident Reporting**: Students and staff can report emergencies with details and optional images
- **Real-time Notifications**: Bell icon with badge showing unread notifications
- **Status Updates**: Departments can update incident status (reported → in progress → resolved)
- **Role-specific Dashboards**: Each user type sees relevant information
- **Admin Panel**: Complete system overview with statistics

## Setup Instructions

### 1. This zip file contains the Virtual Environment only
```bash
source venv\Scripts\activate
```

### 2. Install Dependencies if necessary but they are already included in the virtual environment
```bash
pip install -r requirements.txt
```

### 4. run the project
```bash
python manage.py runserver
```
